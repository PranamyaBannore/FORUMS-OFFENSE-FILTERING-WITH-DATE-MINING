*Requires the Python version 3.7.3 or greater.

//Setup
1. Open the terminal or cmd window at the location of this file.
2. Execute the command "pip install -r src\requirments.txt"
3. After the installations of all the libraries, run Flask App with command "python main.py" to configure the app.
4. It takes few minutes for the data mining algorithm to scan all the links and the recursions. Wait for the message "* Running on http://0.0.0.0:80/ (Press Ctrl+C to quit)".
5. Now, visit "localhost" URL in the browser if everything works as expected you should see the webpage dashboard like in ScreenShots folder.
